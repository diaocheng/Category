从服务器加载数据，重写第一个列表并传入参数以修改插件配置，如下传入可选参数key(返回数据标)和请求的表单数据formdata

          Code:

          $("#button-1").click(function(){
	$("#category-1").category("getData",{
		url:"test1.json",
		key:"category",
		formdata:{
			test:"this is a test formdata"
		}
	});
});
